# **15. Dark Mode Implementation**

### **Dark Mode Color System**

The dark mode uses the same color structure as light mode, with adjusted values for optimal contrast and readability on dark backgrounds.

#### **Key Differences from Light Mode:**

* **Text Colors**: Use white (`rgba(255, 255, 255, ...)`) instead of black
* **Background**: `#121212` (main) / `#222222` (panels)
* **Border**: `rgba(255, 255, 255, 0.08)` for subtle borders
* **Primary Opacity**: 30% and 14% instead of 20% and 5% for better visibility
* **Status Colors**: Same base colors with adjusted opacity (30% and 14% instead of 20%/30% and 5%/9%)

#### **Implementation:**

Dark mode is activated using the `.dark` class on the root element. All color variables automatically switch when dark mode is active.

#### **Usage in Code:**

```tsx
// Tailwind classes automatically adapt
<div className="bg-panel text-foreground">
  <p className="text-text-87">Primary text</p>
  <p className="text-text-60">Secondary text</p>
</div>

// CSS variables automatically switch
<div style={{ backgroundColor: 'var(--color-bg-panel)' }}>
  Content
</div>
```
